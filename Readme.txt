Linguagem C para Intel 8088.

Este trabalho compreende a implementação de um compilador, em suas principais fases, para a linguagem alvo assembly, do processador Intel 8088. Neste trabalho, foram utilizados os softwares FLEX – The Fast Lexical Analyzer – e Bison para gerar os analisadores léxico e sintático, respectivamente.

This work includes the implementation of a compiler in your main phases to the target assembly language, the Intel 8088 processor In this work, the FLEX program was used - The Fast Lexical Analyzer - and Bison to generate the lexical and syntactic analyzers, respectively .
